Sens'it, késako ?

Le Sens'it est un petit objet connecté.
Concrètement il s'agit d'un boîtier en plastique composé de cinq capteurs et d'un bouton utilisant le réseau sans fil SIGFOX.
  SIGFOX est un réseau de communication sans fil à destination des objets connectés.


  • Capteur de température & d'humidité

  • Capteur de lumière

  • Capteur de porte

  • Capteur de vibration

  • Capteur d'aimant



Les données captées

Le Sens'it c'est aussi un site web et une application smartphone qui vous permettront de visualiser les données récoltées par votre Sens'it et crèer des notifications (envoi de sms, email, etc) en fonction de critères particuliers (par exemple température supérieure à 28°C).

Prise en main

Vous serez guidés ici dans la première utilisation de votre Sens'it.




Le contenu de la boîte

La boîte Sens'it contient :

  • un Sens'it
  • Un cable USB pour recharger votre Sens'it
  • Une bande auto-agrippante pour fixer votre Sens'it
  • Un aimant pour le capteur d'aimant
  • Un manuel utilisateur




Mise en route

  • Rendez-vous sur la page web www.sensit.io puis cliquez sur le bouton en haut à droite "démarrer avec mon sens'it".

  • Copiez l'identifiant de votre Sens'it dans la fenêtre qui apparaît. Il s'agit d'un code à six chiffres qui se trouve au dos de votre Sens'it.

  • Appuyez longuement sur le bouton au milieu de votre Sens'it (le rond bleu).

  • La LED centrale s'allumera en vert, indiquant que votre Sens'it est en "mode température".

  • Patientez jusqu'à ce que la LED centrale clignote, ce qui vous indique que le Sens'it envoie les informations (ici la température): celles ci seront disponibles sur l'interface en ligne.




Changer de mode

Le Sens'it ne peut capter qu'un seul type d'information à la fois. Un "mode" correspond à un type de capteur. Pour changer de mode, appuyez de nouveau longuement sur le bouton central.

Chaque mode a une couleur différente

  • Vert, pour le capteur de température & d'humidité
  • Jaune, pour le capteur de lumière
  • Bleu clair, pour le capteur de porte
  • Bleu foncé, pour le capteur vibration
  • Rose, pour le capteur d'aimant



Trucs & astuces

  • Deux pressions rapides sur le bouton central permettent d'envoyer une "alerte" que vous retrouverez dans l'interface en ligne
  • Le Sens'it est limité à 8 messages par heure
  • Lorsque la batterie est vide, la LED principale clignotera deux fois.

Les données

Observez et utilisez les données envoyées par votre Sens'it !

Un label "mode actif" et "inactif" est présent sur chaque modes vous permettant de savoir quel mode est actif. Le mode actif du Sens'it est mis en valeur. Les autres sont "grisés", en dessous.
  Un autre moyen de savoir quel est le mode actuel du Sens'it est d'appuyer une seule fois rapidement sur le bouton central. Le Sens'it clignotera dans la couleur du mode correspondant.




Analyser les données

En cliquant sur le lien "voir détails" en bas à droite de chaque mode, vous accéderez à une interface spécifique au capteur / mode.

Cette interface vous propose des statistiques en fonction du temps des données remontées par votre Sens'it.

Les notifications

Soyez averti dès que votre Sens'it remonte une information !

En cliquant sur le bouton "Créer une notification", vous aurez la possibilité de crèer une notification pour le mode dans lequel vous vous trouvez. Les notifications sont toujours rattachées à un mode.




Paramétrer une notification

Tout les champs sont obligatoires ici. Vous pouvez créer plusieurs notifications par mode. Vous aurez aussi la possibilité de les supprimer.
